To install Git on a local machine, you generally follow these steps:
Download Git: Visit the official Git website (https://git-scm.com/) and download the appropriate version for your operating system (Windows, macOS, or Linux).
Install Git:
For Windows:
Run the downloaded installer.
Follow the installation wizard, selecting your preferred settings.
Ensure to select the option to "Use Git from the Windows Command Prompt" or another preferred terminal.
For macOS:
Run the downloaded .dmg file.
Follow the installation instructions, typically involving dragging the Git icon to your Applications folder.
For Linux:
Use the package manager specific to your distribution. For example:
On Ubuntu/Debian: sudo apt-get install git
On Fedora: sudo dnf install git
Verify Installation: 
Open a terminal or command prompt and run the following command to verify the Git installation:

git --version

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
